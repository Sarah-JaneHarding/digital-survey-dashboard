# Digital Survey Dashboard

This repository contains the source code for the Digital Survey Dashboard application. It is designed to help users create and manage digital surveys easily.

## Features
- User authentication
- Survey creation and management
- Data analysis

## Installation
To install the application, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/digital-survey-dashboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd digital-survey-dashboard
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
